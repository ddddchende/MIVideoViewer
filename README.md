# 小米摄像头录像查看器

一个用于查看小米摄像头录像的网页应用，支持时间线浏览、视频预览和播放功能。

## 功能特性

- 自动扫描并显示摄像头列表
- 时间线可视化显示所有录像
- 支持按日期查看或查看全部时间线
- 视频预览功能（悬停显示静帧）
- 流畅的视频切换（无黑屏）
- 播放控制：播放/暂停、倍速播放（1x-16x）
- 全屏模式支持
- 时间线缩放（1x-128x）和平移
- 可配置视频目录路径

## 安装

### 前置要求

- Node.js 14+
- npm

### 安装步骤

1. 克隆或下载项目到本地

2. 安装依赖
```bash
npm install
```

3. 配置视频目录

编辑 `config.json` 文件，设置你的小米摄像头视频目录路径：
```json
{
  "videoBasePath": "X:\\xiaomi_camera_videos",
  "port": 3000
}
```

或者在网页中点击右上角设置按钮进行配置。

## 使用方法

### 启动服务器

**方式一：使用批处理文件（Windows）**
```bash
start.bat
```

**方式二：使用 Python 脚本**
```bash
python start.py
```

**方式三：使用 npm**
```bash
npm start
```

### 访问应用

启动后在浏览器中访问：http://localhost:3000

### 操作说明

1. **选择摄像头**：页面加载后会自动选择第一个摄像头
2. **查看模式**：
   - 按日期查看：选择特定日期查看录像
   - 全部时间线：查看所有录像的时间线
3. **时间线操作**：
   - 鼠标悬停：显示预览和时间
   - 点击：播放对应视频
   - 滚轮：缩放时间线
   - 滑动条：平移时间线
4. **播放控制**：
   - 空格键：播放/暂停
   - F 键：切换全屏
   - 双击视频区域：进入全屏
5. **倍速播放**：支持 1x、2x、4x、8x、16x 倍速

## 视频目录格式

应用期望的视频目录结构如下：

```
视频根目录/
├── 摄像头名称1/
│   ├── YYYYMMDDHH/           # 日期时间文件夹
│   │   ├── MMMSS_timestamp.mp4   # 视频文件
│   │   └── ...
│   └── ...
├── 摄像头名称2/
│   └── ...
└── ...
```

例如：
```
X:\xiaomi_camera_videos\
├── camera_living_room\
│   ├── 2024031508\
│   │   ├── 00M00S_1710489600000.mp4
│   │   ├── 01M00S_1710489660000.mp4
│   │   └── ...
│   └── 2024031509\
│       └── ...
└── camera_bedroom\
    └── ...
```

## 配置说明

| 字段 | 说明 | 默认值 |
|------|------|--------|
| videoBasePath | 视频目录路径 | X:\xiaomi_camera_videos |

## 技术栈

- 后端：Node.js + Express
- 前端：原生 HTML/CSS/JavaScript
- 视频播放：HTML5 Video API
- 时间线渲染：Canvas

## 常见问题

### 1. 视频无法播放

- 检查视频目录路径是否正确
- 确认视频文件格式为 MP4
- 查看浏览器控制台是否有错误信息

### 2. 摄像头列表为空

- 确认视频目录下有摄像头子文件夹
- 检查文件夹命名是否符合格式要求

### 3. 时间线显示异常

- 刷新页面重试
- 检查视频文件命名是否符合格式
